# PinkyPromise
2020 iOS Project

> 약속을 지키지 못하는 모두에게,
> 캘린더를 통해 약속 지킴 상황을 확인하고, 나 또는 친구와의 약속을 만들 수 있는 어플입니다.

## 개발 환경

Xcode Version 11.3 <br>
Swift Version 5

| 로그인(Google,Apple)| O | X | 이현재 | 진행 중 |
